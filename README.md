## About The Project

Proyek ini merupakan aplikasi web Survei Kepuasan Pelanggan yang dikembangkan untuk mengumpulkan, mengelola, dan menganalisis tingkat kepuasan pelanggan secara efektif. Aplikasi dibangun menggunakan Laravel 11 sebagai framework utama, Tailwind CSS untuk antarmuka yang modern dan responsif, Laravel Jetstream untuk autentikasi dan manajemen pengguna, serta ApexCharts untuk visualisasi data hasil survei dalam bentuk grafik yang informatif.
This project is a web-based Customer Satisfaction Survey application developed to collect, manage, and analyze customer satisfaction levels effectively.
The application is built using Laravel 11 as the main framework, Tailwind CSS for a modern and responsive user interface, Laravel Jetstream for authentication and user management, and ApexCharts for visualizing survey results through informative charts.
